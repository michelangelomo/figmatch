Manual Installation

Since we're too exclusive for the community plugins (we have standards)

1

Download the Plugin

Get the latest Figmatch plugin file from our exclusive repository. It's like getting into a secret design speakeasy.

https://figmatch.love/download/plugin.zip
2

Extract the Files

Unzip the downloaded file to a memorable location. We recommend a folder called "Love" on your desktop for maximum visibility.

figmatch-plugin-v1.0/
3

Open figma Desktop

Launch the figma desktop app. The web version won't work for local plugins - we're too sophisticated for browsers.

Plugins ‚Üí Development ‚Üí Import plugin from manifest...
4

Select manifest.json

Navigate to the extracted folder and select the manifest.json file. This is where the magic begins.

figmatch-plugin-v1.0/manifest.json
5

Run the Plugin

Go to Plugins ‚Üí Development ‚Üí Figmatch and launch it. Your love life is about to get a serious upgrade.

Plugins ‚Üí Development ‚Üí Figmatch ‚Üí Run
6

Start Swiping

Create your profile, set your design preferences, and start matching. Remember: good kerning is a green flag.

Profile ‚Üí Preferences ‚Üí Start Matching
